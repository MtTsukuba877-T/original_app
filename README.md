## 1. サービス概要（※最後に書いてOK）

- このサービスはどんなサービスか（3行以内）
  -  契約企業の従業員向けのストレスチェックサービス。特に、工場や病院、介護事業所など、企業から個々の従業員に固有のメールアドレスが付与されていない企業の従業員向け。

- 誰の、どんな課題を、どう解決するか
  -  ストレスチェックは、以下の課題を解決します。
　-  契約企業の従業員がストレスチェックを定期的に受検することにより、自身のストレス状態を自覚し、ストレス過多によるメンタル不調に陥る前に対処・相談することを促します。
　・契約企業が、集団分析結果を基に職場改善につなげ、従業員がメンタル不調になることを防止し、かつ、生産性向上を目指すことを促します。

## 2. このアイデアはどこから生まれたか

### 2-1. きっかけとなった体験・感情

このアイデアの元になった
**自分自身の体験・感情・違和感**を教えてください。

- いつ
- どんな状況で

できるだけ具体的なエピソードで書いてください。
-  ストレスチェック（Web受検）は、ログインするためのID・パスワードを各受検者のメールアドレスに通知して行うことが一般的です。しかし、上記で挙げた、工場勤務の方や、病院の病棟で勤務する看護師、介護事業所で勤務する介護スタッフ、飲食店などの店舗で接客をしている従業員は、個々にメールアドレスを勤務先から付与されていないことが多く、ID・パスワードを通知することができず、ID＝従業員ID、パスワード＝生年月日YYYYMMDDなどで設定することが多い状況です。加えて、受検者個々にメールアドレスが設定されていないため、受検期間中にメールを配信して受検を促す（受検勧奨）することもできず、企業の管理者が受検を促すことに大きな負担を伴っている状況です。
よって、この状況を改善するアプリを制作したいと思いました。

### 2-2. なぜそれが「気になった」のか

同じ体験をしても、気にしない人もいます。
**なぜ自分はそれが気になったのだと思いますか？**

- 自分の性格
- 過去の経験
- 大切にしている価値観

など結びつけて考えてみてください。

-  ストレスチェックのような「セルフチェック」は、状態の悪い人ほど受けたがらない傾向があり、このような方々こそ、きちんとフォローする必要があるのですが、現状ではよく行き届いているとは言えない状況であるためです。
-  ストレス過多からメンタル不調に陥る傾向があることは良く知られており、ストレスチェックを受検することにより、定期的に自身のストレス状況を把握し、必要に応じてストレスを解消する取り組み（セルフケア）をしたり、医師やカウンセラーなどの専門家に相談することを促すことで、メンタル不調の早期発見・早期対応、そして、早期の回復を目指すことが重要だと考えています。
-  私が以前に勤務した職場の上司に、仕事熱心で毎月90～100時間程度残業していた方がいましたが、最終的に出勤が不安定になり（朝起きられなくなり）、体調をくずす結果となりました。
-  このような経験もあり、職場で働く方々のメンタルヘルスの向上に役立つストレスチェックを推進していきたいと考えています。
